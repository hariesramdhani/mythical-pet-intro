Mythical Pet is a chatbot that lives inside LINE and it allows the user to own virtual pet in LINE without the need to download the additional application (Imagine a Tamagotchi but with more sophisticated features). With Mythical Pet chatbot user can evolve, feed and even couple their pets.

After a while group chat can become boring, Mythical Pet is here to color the group chat with features that increase user interaction in a group chat, not to mention that Mythical Bot can also be used by the solo user whenever s/he feels bored.

3 EASY WAYS TO START

Starting is easy, invite the pet and you're ready to play!

  • Invite the bot by scanning the QR code using your LINE APP
  • Type ‘Hello <your_pet_name>’
  • Type ‘qr’, the menu will pop up and you’re ready to play

Even with the complex features, the chatbot is made as simple as possible so it will easily be digested by the user. The usage of carousels, flex boxes and the digital images that were drawn are optimized (not too much and not too less). This will allow the bot to work faster and keep the data cost lower (considering that the user wont always be in a wi-fi-present environment). The backend code for the chatbot is also optimized to make the bot processes as fast as possible.
